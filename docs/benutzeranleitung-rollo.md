# Rollo-Steuerung – für die Familie

So steuert ihr die Rollos im Haus.

## Per Sprache (Siri)

„Hey Siri, …" sagen – aber nur zuhause im WLAN!

| Sagen | Passiert |
|---|---|
| „Wohnzimmer Rollos hoch" | Wohnzimmer ganz öffnen |
| „Wohnzimmer Rollos runter" | Wohnzimmer ganz schließen |
| „Küche Rollos 30%" | Küche auf 30% |
| „Alle Rollos stopp" | Alle anhalten |
| „Erdgeschoss beschatten" | EG auf Halbschatten |
| „Westen nicht mehr beschatten" | Westseite wieder öffnen |

## Per WhatsApp oder Telegram

Schreibt Hermes einfach, was ihr wollt:

- „Mach die Rollos im Wohnzimmer runter"
- „Erdgeschoss beschatten"
- „Küche Jalousien 50%"
- „Alle Rollos hoch"
- „Osten nicht mehr beschatten"

Hermes versteht das und führt es aus.

## Welche Räume gibt es?

| Wenn ihr sagt… | …betrifft das |
|---|---|
| **Küche** | Beide Küchenfenster |
| **Wohnzimmer** | Beide Wohnzimmerfenster |
| **Schlafzimmer** | Alle drei Schlafzimmer |
| **Arbeitszimmer** | Beide Arbeitszimmer |
| **Gästebad** | Das Gästebad |
| **Badezimmer** | Das Badezimmer oben |
| **Till** | Tills Zimmer |
| **Jan** | Jans Zimmer |

## Ganze Etagen auf einmal

| Sagen | Betrifft |
|---|---|
| **Erdgeschoss** / **EG** | Küche, Gästebad, Wohnzimmer |
| **Obergeschoss** / **1. OG** | Schlafzimmer, Arbeitszimmer, Badezimmer |
| **Dachgeschoss** / **2. OG** | Till, Jan |
| **Alle** | Das ganze Haus |

## Nach Himmelsrichtung

| Sagen | Betrifft |
|---|---|
| **Osten** | Morgensonne-Seite (Küche, Gästebad, Bad, Arbeitszimmer) |
| **Westen** | Abendsonne-Seite (Wohnzimmer, Schlafzimmer) |
| **Norden** | Nordseite (Küche, Arbeitszimmer, Till, Jan) |

## Was kann ich sagen?

| Befehl | Was passiert |
|---|---|
| **auf**, **hoch**, **öffnen** | Ganz öffnen |
| **zu**, **runter**, **schließen** | Ganz schließen |
| **beschatten**, **Sonnen­schutz** | Auf Halbschatten (50%) |
| **stop**, **halt** | Anhalten |
| **nicht mehr beschatten** | Beschattung beenden (öffnen) |
| Eine Zahl, z.B. **30**, **75** | Genau auf diese Prozentzahl |

## Beispiele

| Sagen | Ergebnis |
|---|---|
| „Wohnzimmer Rollos runter" | Wohnzimmer schließt |
| „Erdgeschoss beschatten" | EG auf Halbschatten |
| „Küche 30%" | Küche auf 30% |
| „Alle Rollos stopp" | Alles anhalten |
| „Westen nicht mehr beschatten" | Westseite öffnet |
| „Osten Rollos hoch" | Ostseite öffnet |

## Was die Automatik macht

Die Rollos fahren von selbst – ihr müsst nichts tun.

- **Morgens** wenn die Sonne stark scheint: Ostseite schattet ab
- **Nachmittags**: Westseite schattet ab
- **Abends**: Alle Rollos schließen
- **Nachts**: Außenlicht kurz an, dann aus

**Wichtig:** Wenn ihr die Rollos manuell verstellt (per Sprache oder Schalter), bleibt das so. Die Automatik überschreibt eure Änderung nicht.
