# Gardena Bewässerung

## Ziel

Gardena-Bewässerung überwachen und, falls möglich, steuern.

## Benötigte Informationen

- Gardena Smart System vorhanden?
- Gateway/Hub vorhanden?
- Welche Zonen/Ventile gibt es?
- Zugang über Gardena-Konto/API möglich?

## Geplanter Ansatz

1. API-/Integrationsweg prüfen.
2. Status und Zonen erfassen.
3. Manuelle Steuerung über Node-RED/Hermes ermöglichen.
4. Wetter-/Regenlogik später ergänzen.
