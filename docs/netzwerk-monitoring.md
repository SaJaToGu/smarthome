# LAN/WLAN Monitoring und Verbesserung

## Ziel

Heimnetz überwachen, Ausfälle sichtbar machen und WLAN/LAN schrittweise verbessern.

## Erste Werkzeuge

- Uptime Kuma für Verfügbarkeit wichtiger Geräte
- Grafana optional für Langzeitwerte
- FritzBox/Router-Auswertung später ergänzen

## Nächste Schritte

1. Wichtige Geräte inventarisieren: Router, NAS, Raspberry Pi, Shellys, Drucker, Access Points.
2. Uptime-Kuma-Monitore anlegen.
3. IP-/Namensschema dokumentieren.
4. WLAN-Schwachstellen systematisch messen.
